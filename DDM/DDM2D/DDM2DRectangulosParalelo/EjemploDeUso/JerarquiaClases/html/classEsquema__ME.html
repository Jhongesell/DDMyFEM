DescripciÃ³n detallada

Clase base para definir el Esquema Maestro-Esclavo en MPI.

Clase base para definir el Esquema Maestro-Esclavo para programar en paralelo mediante el paso de mensajes usando MPI, donde el primer procesador (id = 0) es el nodo mestro y el resto son los nodos esclavos. Las tareas se pueden repartir de manara que subdominios contiguos queden en un mismo nodo esclavo o queden en distinto nodo esclavo.
