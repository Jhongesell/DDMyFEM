Descripci√≥n detallada

Clase para definir el Esquema Maestro-Esclavo del mÈtodo de descomposiciÛn de dominio de subestructuraciÛn (Complemento de Schur)

Clase para definir el Esquema Maestro-Esclavo para programar en paralelo mediante el paso de mensajes usando MPI, donde el primer procesador (id = 0) es el nodo mestro y el resto son los nodos esclavos. Las tareas se reparten de una a una entre los nodos esclavos, si hay mas tareas que procesadores se asignaran mas de una tarea a cada esclavo, el reparto no necesariamente es homogeneo.
